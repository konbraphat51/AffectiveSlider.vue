# Affective Slider Vue - Demo

This folder contains the built demo application for GitHub Pages.

**Live Demo**: https://konbraphat51.github.io/AffectiveSliderVue/

## Building

To rebuild this demo:

```bash
npm run build:demo
```

This will build the demo application from the source files and output to this `/docs` folder.

## GitHub Pages Setup

To enable GitHub Pages for this repository:

1. Go to repository Settings
2. Navigate to Pages section
3. Under "Build and deployment"
   - Source: Deploy from a branch
   - Branch: Select your main branch
   - Folder: `/docs`
4. Click Save

GitHub Pages will automatically deploy the contents of this folder to:
https://konbraphat51.github.io/AffectiveSliderVue/

## What's Included

- `index.html` - Main HTML file
- `assets/` - Bundled JavaScript and CSS files
- `images/` - Required PNG images for the sliders
